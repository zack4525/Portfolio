헬씈 - Well-being Comunity Details

Project information

  • Motive:
    건강한 삶을 위해 BMI에 맞춘 맞춤형 식단,
    운동 제공 커뮤니티
  • Language:
    React.js, CSS3, SASS, JavaScript, Node.js , MySql, MongoDB
  • Project date:
    2021.04.22~ 5.06 (15일)
  • Frontend Modules:
    axios, @material-ui, antd, chart.js, formik, http-proxy-middleware, moment, jwt-decode, react, react-chartjs-2, react-dom, react-file-base64, react-quill, redux react-redux, redux-promise, redux-thunk, styled-components, web-vitals, yup
  • Backend Modules:
    bcrypt, body-parser, cookie-parser, dotend, express, jsonwebtoken, moment, mongoose, mysql2

Details

로그인 기능

- Token과 Cookie를 이용한 인증 방식
- 인증된 정보는 Redux로 상태관리

Main Page

- Formik과 Yup 라이브러리를 이용하여 유효성 검사
- 서버 측 오류는 아이디 중복 제외 없음
- 키, 몸무게로 BMI 측정
- BMI에 따른 추천 운동 영상 제공 (iframe 태그 사용)
- BMI와 시간에 따른 랜덤 식단 제공
- 자유 게시판 조회수 Top5 게시글 제공

자유게시판

- REST API, MySql을 이용하여 게시글 및 댓글 구현
- Quill 라이브러리로 게시판 작성페이지 구현
- Quill 라이브러리로 태그까지 같이 저장된 게시글을 'dangerouslySetInnerHTML' 속성 이용하여 작동 구현

운동 인증 게시판

- REST API을 이용하여 MongoDB, MySql를 하이브리드로 게시글 및 좋아요 구현
- react-file-base64를 이용하여 이미지 파일 구현
- 게시글의 '_id' 값을 MySql로 연동하여 좋아요 구현 및 중복 확인 구현
- Redux 상태 값으로 글 삭제, 글 수정 허용

MyPage

- Rest API, MongoDB로 몸무게 변경 구현
- 변경된 몸무게는 MySql로 저장 후 최근 15개의 몸무게 변화 차트 제공
- Rest API, MySql로 자신이 작성한 자유게시판 게시글 제공
