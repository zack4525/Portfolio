Details

YAMA COIN API

- BackEnd 서버와, YAMA COIN 서버의 RPC 통신을 통하여 구현
- 회원가입 시 USER ID로 "getnewaddress"하여 DB에 저장
- 로그인 시 'getbalance'를 이용하여 DB의 데이터와 실제 보유코인 갯수를 비교
- 거래 성사 시 'sendfrom'를 이용하여 바로 YAMA COIN이 거래되도록 구현

Main Page

- Latest Blocks은 'LatestBlocks'로 마지막 블록 높이를 구한 후 'getbestblockhash' 최근 5개 블록 해쉬값을 보여준다
- Latest Transactions은 'listtransactions'을 이용하여 해시값이 같은 transaction을 제외한 최근 5개 txid를 보여준다
- Login은 backend mysql로 연동 후 redux state 값 변경
- Logout을 하지 않고 클라이언트 종료 후 다시 접속 시 남아있는 cookie로 자동 로그인 구현
- '블록 검색'은 'getblockhash'을 이용하여, block height, block hash값으로 검색 구현
- '지갑 주소로 보내기'는 'sendfrom' 을 이용하여 보내기 구현

거래소

- 'Upbit' API와 'react-google-charts'를 이용하여 실시간 차트 구현
- YAMA COIN을 실제 거래소와 같이 거래되게 구현
- Yama COIN 거래내역과 'react-google-charts'를 이용하여 차트 구현
- 'Upbit' API를 이용하여 실시간 시세 조회 가능

Block#

- 'getblock'으로 Block에 담긴 정보를 보여준다

Transaction#

- 'getrawtransaction'으로얻은 해쉬값을 'decoderawtransaction'를 이용하여 거래 정보를 보여준다

회원가입

- 정규표현식과 axios를 이용하여 실시간 ID 중복검사 및 규격에 맞추었다
- background를 동영상으로 이쁘게 꾸며봤다
