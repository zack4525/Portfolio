Details

로그인 기능

- 'passport', 'passport-kakao' 모듈을 사용하여 KAKAO API 로그인 구현


실시간 시세 제공

- 네이버 금융 Top50, KOSPI 지수를 실시간 크롤링을 통하여 구현

차트 제공

- 서버가 평일 09 ~ 16시 사이 10분 단위(node-schedule 모듈)로 시세를 크롤링 하여 DB에 저장하도록 구현
- 카카오톡 채팅창 구현 (형식만 카카오톡)
- 종목명 클릭 이벤트로 1일, 3일, 1주일치 데이터를 서버에 AJAX로 요청하여 Chart.js로 구현
- KOSPI Top50 종목이 변경될 가능성이 있으므로, Top60까지 DB에 저장하도록 구현

구매

- '구매' 클릭 시 BootStrap Modal 이용하여 구매창 구현
- '구매' 클릭 시 JavaScript 이용하여 Modal 내용 변경
- 구매 시 잔여 포인트 차감 및 포인트 부족 시 error 메세지 호출

MyPage

- 구입 시 가격과, 현재 가격을 크롤링 하여 비교 한 후 수익률 제공
- 현재까지 충전한 전체 금액과 보유중인 주식의 현재 총액을 비교하여 전체 수익 제공
- '매도' 클릭 시 (현재 금액 * 보유 수량)만큼 포인트 상승

오류

- 로그아웃 기능 구현 실패
- 세션이 고장나서 로그인 시 다른 사이트에서도 로그인 되어있음 ㅠㅠ
