헬씈 - Well-being Comunity


제작 동기 : 건강한 삶을 위해 BMI에 맞춘 맞춤형 식단, 운동 제공 커뮤니티


제작 기간 : 2021.04.22~ 5.06 (15일)


사용 언어 : React, CSS3, SASS, JavaScript, Node.js , MySql, MongoDB


사용 모듈 :
FrontEnd : axios, @material-ui, antd, chart.js, formik, http-proxy-middleware, moment,
jwt-decode, react, react-chartjs-2, react-dom, react-file-base64, react-quill, redux
react-redux, redux-promise, redux-thunk, styled-components, web-vitals, yup

BackEnd : bcrypt, body-parser, cookie-parser, dotend, express, jsonwebtoken, moment, mongoose, mysql2


기능 설명


로그인 기능


- Token과 Cookie를 이용한 인증 방식

- 인증된 정보는 Redux로 상태관리


메인 페이지

- Formik과 Yup 라이브러리를 이용하여 유효성 검사

- 서버 측 오류는 아이디 중복 제외 없음

- 키, 몸무게로 BMI 측정

- BMI에 따른 추천 운동 영상 제공 (iframe 태그 사용)

- BMI와 시간에 따른 랜덤 식단 제공

- 자유 게시판 조회수 Top5 게시글 제공


자유게시판

- REST API, MySql을 이용하여 게시글 및 댓글 구현

- Quill 라이브러리로 게시판 작성페이지 구현

- Quill 라이브러리로 태그까지 같이 저장된 게시글을 'dangerouslySetInnerHTML' 속성 이용하여 작동 구현


운동 인증 게시판

- REST API, MongoDB, MySql을 이용하여 게시글 및 좋아요 구현

- react-file-base64를 이용하여 이미지 파일 구현

- 게시글의 '_id' 값을 MySql로 연동하여 좋아요 구현 및 중복 확인 구현

- Redux 상태 값으로 글 삭제, 글 수정 허용


마이 페이지

- Rest API, MongoDB로 몸무게 변경 구현

- 변경된 몸무게는 MySql로 저장 후 최근 15개의 몸무게 변화 차트 제공

- Rest API, MySql로 자신이 작성한 자유게시판 게시글 제공


내가 맡은 파트

전체 BackEnd 구축, 자유게시판, 마이페이지

ARENA


제작 동기 : 주식 모의투자 웹페이지


제작 기간 : 2021.03.08~ 3.24 (17일)


사용 언어 : EJS, CSS3, JavaScript, Node.js , MySql


사용한 라이브러리 : J-Query(ajax), BootStrap, chart.js


사용 모듈 : axios, cheerio, cookie-parser, debug, dotenv, ejs
express, express-session, iconv, jschardet, moment, morgan,
mysql2, node-schedule, passport, passport-kakao, requset, requset-promise


기능 설명


로그인 기능


- passport, passport-kakao 모듈을 사용하여 KAKAO 로그인 구현


실시간 시세 제공

- 네이버 금융 Top50, KOSPI 지수를 실시간 크롤링을 통하여 구현


차트 제공

- 서버가 평일 09 ~ 16시 사이 10분 단위(node-schedule 모듈)로 시세를 크롤링 하여 DB에 저장하도록 구현

- 종목명 클릭 이벤트로 1일, 3일, 1주일치 데이터를 서버에 AJAX로 요청하여 Chart.js로 구현

- KOSPI Top50 종목이 변경될 가능성이 있으므로, Top60까지 DB에 저장하도록 구현


충전 기능

- form 태그로 서버와 통신


구매

- '구매' 클릭 시 BootStrap Modal 이용하여 구매창 구현

- '구매' 클릭 시 JavaScript 이용하여 Modal 내용 변경

- 구매 시 잔여 포인트 차감 및 포인트 부족 시 error 메세지 호출


마이 페이지

- 구입 시 가격과, 현재 가격을 크롤링 하여 비교 한 후 수익률 제공

- 현재까지 충전한 전체 금액과 보유중인 주식의 현재 총액을 비교하여 전체 수익 제공

- '매도' 클릭 시 (현재 금액 * 보유 수량)만큼 포인트 상승
