Details

로그인 기능

- User DB와 연동, 입력 정보와 DB정보 일치 시 로그인 후 세션에 ID값 저장
- 로그인 후 메인페이지 이동 가능하게 구현

회원 가입

- JavaScript로 비밀번호,비밀번호 확인 여부, 이메일 양식 정규표현식 적용


메인 페이지

- 좌, 우 버튼 클릭 시 사진 1개 크기만큼만 스크롤 이동 (JavaScript)
- 최근 업로드 된 50개의 게시글만 로딩되게 설정
- BootStrap Modal을 이용하여 사진 클릭시 Modal창 띄움
- 사진 클릭 시 JavaScript로 Modal창 정보 변경
- 사진 클릭 시 AJAX로 해당 게시물 댓글 로딩 구현

게시글

- 게시글 작성자와 로그인된 ID일치 시에만 게시글 삭제되도록 구현
- 댓글 작성자와 로긍니된 ID일치 시에만 댓글 삭제되도록 구현

MyPage

- 로그인된 ID의 게시글만 노출되게 구현
- 게시글 보기 Modal창 구현(BootStrap)
- 사진 UpLoad는 Modal창으로 구현

사진 업로드

- 업로드 된 사진 JavaScript 이용하여 미리보기 제공
- 사진 업로드는 DB가 아닌 호스팅 서버에 저장('호스팅 서버에 저장 된 주소+파일명'으로 DB 저장)
- 저장 과정에서 한글 깨짐현상 -> iconv 이용 UTF-8 적용
